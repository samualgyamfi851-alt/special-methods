# special-methods
my special method

__add__() – Defines how the addition (+) operator behaves for objects of a class.
__sub__() – Defines how the subtraction (-) operator behaves for objects of a class.
__mul__() – Defines how the multiplication (*) operator behaves for objects of a class.
__getitem__() – Allows an object to be accessed using indexing, such as object[index].
__setitem__() – Allows values in an object to be modified using indexing, such as object[index] = value.
__contains__() – Defines how the in operator checks whether an item exists within an object.
__call__() – Allows an object to be called like a function using parentheses.
__iter__() – Returns an iterator object, making the object usable in a for loop.
__next__() – Returns the next item in an iteration and raises StopIteration when there are no more items.
__bool__() – Determines whether an object evaluates to True or False in Boolean expressions.
__del__() – Called automatically when an object is about to be destroyed, allowing cleanup of resources if necessary
